วิธีติดตั้ง:

  1. เปิด Developer Mode และ USB Debugging บนมือถือ Android
  2. คัดลอก APK ไปยังมือถือ หรือใช้ ADB ติดตั้ง
  3. เปิดไฟล์ APK บนมือถือและกด Install
  4. เปิดแอป Photo Gallery และทดสอบการถ่ายรูป
